# Azure Virtual Machine Deployment Project

## Overview

This project demonstrates the deployment and configuration of a Linux Virtual Machine in Microsoft Azure as part of my Azure Administrator (AZ-104) hands-on learning journey.

## Project Objectives

- Deploy a Virtual Machine in Azure
- Configure networking resources
- Implement secure SSH authentication
- Manage resources using Azure Resource Groups
- Gain practical cloud administration experience

## Resources Created

- Virtual Machine (Ubuntu 24.04 LTS)
- Resource Group (RG-Carol-Lab)
- Virtual Network (VNet)
- Network Security Group (NSG)
- Public IP Address
- Network Interface

## Skills Demonstrated

- Microsoft Azure Administration
- Virtual Machine Deployment
- Azure Networking
- Resource Management
- Linux Administration
- SSH Authentication
